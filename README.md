# SmartCode — Информационная система для IT-школы

Полноценный веб-проект с системой авторизации, ролевой моделью, личными кабинетами, кошельком и админ-панелью.

---

## О проекте

SmartCode — это информационная система для управления IT-школой. Проект разработан в рамках учебной практики для демонстрации навыков веб-разработки на PHP и MySQL.

### Основные возможности

- **Регистрация, авторизация, восстановление пароля** через секретное слово
- **3 роли пользователей:** студент, преподаватель, администратор
- **Личные кабинеты** с разным функционалом для каждой роли
- **CRUD** для курсов, мероприятий и пользователей (админка)
- **Запись на платные курсы** (списание с баланса) и бесплатные мероприятия
- **Кошелёк** с возможностью пополнения и историей транзакций
- **Тёмная тема** с сохранением в localStorage
- **Модальные окна** через AJAX-запросы
- **Адаптивный дизайн** под все устройства

## Цель

Показать навыки:

- Разработка полноценного веб-приложения на чистом PHP
- Работа с MySQL (JOIN, подзапросы, подготовленные запросы)
- Клиент-серверная архитектура с ролевой моделью
- Вёрстка интерфейсов с адаптивом и тёмной темой
- Взаимодействие с сервером через AJAX и JSON
- Безопасность (защита от SQL-инъекций, хеширование паролей)

---

## Использованные технологии

### Frontend:
- **HTML5** — семантическая разметка
- **CSS3** — Flexbox, Grid, медиа-запросы, анимации
- **JavaScript** — AJAX, работа с DOM, localStorage
- **Font Awesome** — иконки
- **Google Fonts** — шрифт Montserrat

### Backend:
- **PHP** — процедурный стиль (базовый уровень)
- **MySQL** — база данных, JOIN, индексы
- **AJAX** — асинхронные запросы без перезагрузки страницы

### Инструменты:
- **OpenServer** / **XAMPP** — локальный сервер
- **Git** и **GitHub** — контроль версий
- **VS Code** — среда разработки
